Design notes

How this site was rebuilt, and why.

This page once held five exploratory visual directions. It now records the decisions behind the implemented site — a short rationale, kept for reference.

Rationale

Decisions behind the redesign.

The goal: a calm, recruiter-first portfolio that itself demonstrates clear delivery thinking — structure over decoration.

Direction
Light editorial. Warm paper background, near-black ink, one accent, and generous whitespace. Recruiter-first, legible, and built to age well.
Concept
The delivery-system idea stays as the structural spine — signal, intervention, outcome — instead of a literal control-room dashboard that crowded the original hero.
Typography
Inter carries content; IBM Plex Mono carries labels, section indices, and metrics — a precise annotation layer that reads as systematic.
Colour
A single clay accent, used sparingly on key numbers, marks, and active states. It replaces the earlier four-colour palette that read as muddy.
Rhythm
One dark Approach band and a dark footer; everything else is light, separated by hairline rules rather than heavy boxes.
Motion
A subtle reveal-on-scroll, fully disabled under reduced-motion settings, with all content visible if JavaScript fails to load.
Build
Plain HTML and CSS with one small JavaScript file. No build step; the site deploys as static files.
